# Aplicación Interactiva de Algoritmos Ráster

## Descripción

Este proyecto consiste en una aplicación gráfica desarrollada en C# y Windows Forms para el estudio y visualización de algoritmos ráster utilizados en Computación Gráfica. La herramienta permite representar de manera interactiva el funcionamiento de diferentes algoritmos de líneas, círculos y relleno, mostrando tanto el resultado gráfico como los pasos internos de cálculo.

## Algoritmos Implementados

### Líneas

* DDA (Digital Differential Analyzer)
* Punto Medio
* Curva de Bézier

### Círculos

* Círculo Base
* Método Polar
* Método de Rotación por Matriz

### Relleno

* Flood Fill
* Boundary Fill
* Scanline Fill

## Características

* Interfaz gráfica amigable desarrollada en Windows Forms.
* Visualización de los píxeles generados sobre una cuadrícula cartesiana.
* Representación paso a paso de los cálculos realizados por cada algoritmo.
* Comparación visual entre diferentes métodos de generación gráfica.
* Tablas informativas con los valores calculados durante la ejecución.
* Herramientas de dibujo y relleno para pruebas interactivas.

## Tecnologías Utilizadas

* C#
* .NET Framework 
* Windows Forms

## Objetivo Académico

El proyecto fue desarrollado con fines educativos para comprender el funcionamiento interno de los algoritmos ráster y analizar las diferencias entre distintos métodos de generación de gráficos en computación.
